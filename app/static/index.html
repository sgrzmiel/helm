Set up Google Drive integration

          
        

        
One-time setup. Takes ~10 minutes.

        
    
          
  1. Go to Google Cloud Console → APIs & Services → Credentials.
    2. 
          
  2. Create (or pick) a project. Click + Create credentials → OAuth client ID.
    3. 
          
  3. Application type: Web application.
    4. 
          
  4. Under Authorized redirect URIs, add this exact URL:
    
            http://localhost:8765/api/google/callback
    5. 
          
  5. Save. Copy the Client ID and Client secret Google shows.
    6. 
          
  6. In APIs & Services → Enabled APIs, enable Google Drive API + Google Slides API.
    7. 
          
  7. Paste the Client ID + Client secret into the form here, click Save.
    8. 
          
  8. Click Connect Google Drive and grant access in the Google consent screen.
